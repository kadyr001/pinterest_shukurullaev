# Pinterest KLON (Пинтерест Копиясы)

Классикалык Pinterestтин иштешинин көчүрмөсү — колдонуучулар сүрөттөрдү (пиндерди) бөлүшө алат, доскаларга топтойт, башкалардын пиндерин лайктап, комментарий калтырып, сактай алат. Толугу менен FastAPI (Python) менен жазылган, SQLite маалымат базасы колдонулат.

---

## Мүмкүнчүлүктөр

### 1. Колдонуучулар (Auth)
- Регистрация (username, email, пароль)
- Кирүү (логин) — JWT токен (access + refresh)
- Өз профилиңди көрүү жана жаңыртуу
- Башка колдонуучулардын профилдерин көрүү

### 2. Пиндер (Pin)
- Пиндерди түзүү (сүрөт + аталыш + баяндама + шилтеме)
- Пиндерди доскаларга кошуу
- Пиндерди тегдер менен белгилөө
- Пиндерди оңдоо жана өчүрүү (өзүңдүкүн гана)
- Пиндерге лайк басуу / лайкты алуу
- Пиндерди сактоо / сактаганды алуу

### 3. Доскалар (Board)
- Доскаларды түзүү (жашыруун же коомдук)
- Доскага пиндерди кошуу
- Доскадагы пиндердин тизмеси
- Колдонуучунун бардык доскаларын көрүү

### 4. Лента (Feed)
- Негизги лента (акыркы пиндер)
- Издөө (пиндердин аталышы боюнча)
- Тегдер боюнча чыпкалоо
- Trending (эң көп лайк алган пиндер)

### 5. Башка
- Комментарийлер (пинге жазуу, өчүрүү)
- Саналган көрсөткүчтөр (лайк, комментарий, сактоо саны)
- Сүрөттөрдү жүктөө (jpg, png, gif, webp)

## Колдонулган технологиялар

| Технология | Багыты |
|---|---|
| **FastAPI** | API сервери (Python) |
| **SQLAlchemy 2.0** | ORM, моделдер, маалымат базасы |
| **SQLite** | Маалымат базасы (локалдуу иштөө үчүн) |
| **JWT (python-jose)** | Авторизация, токендер |
| **passlib (bcrypt)** | Паролдорду шифрлөө |
| **sqladmin** | Админ панели (браузерден башкаруу) |
| **Pydantic** | Маалыматтарды текшерүү (Schema) |
| **Pillow** | Сүрөттөрдү иштетүү |

## Орнотуу (Windows / Mac / Linux)

### 1. Репозиторияны көчүрүү

```bash
git clone https://github.com/khamid9/pinterest_klon.git
cd pinterest_klon
```

### 2. Виртуалдык чөйрө түзүү

```bash
python -m venv venv
source venv/bin/activate   # Linux/Mac
venv\Scripts\activate      # Windows
```

### 3. Керектүү китепканаларды орнотуу

```bash
pip install -r requirements.txt
```

### 4. Серверди иштетүү

```bash
uvicorn main:app --reload
```

Сервер иштей баштайт: **http://127.0.0.1:8000**

## Админ панели

SQLAdmin админ панели төмөнкү даректе:
```
http://127.0.0.1:8000/admin
```

Админ панели аркылуу бардык моделдерди (колдонуучулар, пиндер, доскалар, тегдер, комментарийлер, лайктар, подпискалар) башкарууга болот.

## API эндпоинттери

### Auth (`/auth`)
| Метод | Эндпоинт | Баяндама |
|---|---|---|
| POST | `/auth/register/` | Катталуу |
| POST | `/auth/login/` | Кирүү (токен алуу) |
| GET | `/auth/me/` | Өз профилиң |
| PUT | `/auth/me/` | Профилиңди жаңыртуу |

### Пиндер (`/pins`)
| Метод | Эндпоинт | Баяндама |
|---|---|---|
| GET | `/pins/` | Бардык пиндер |
| GET | `/pins/{id}/` | Бир пин |
| POST | `/pins/` | Жаңы пин түзүү |
| PUT | `/pins/{id}/` | Пинди оңдоо |
| DELETE | `/pins/{id}/` | Пинди өчүрүү |
| POST | `/pins/{id}/like/` | Лайк басуу / алуу |
| POST | `/pins/{id}/save/` | Сактоо / алуу |

### Доскалар (`/boards`)
| Метод | Эндпоинт | Баяндама |
|---|---|---|
| GET | `/boards/` | Бардык коомдук доскалар |
| GET | `/boards/{id}/` | Бир доска |
| POST | `/boards/` | Жаңы доска түзүү |
| PUT | `/boards/{id}/` | Досканы оңдоо |
| DELETE | `/boards/{id}/` | Досканы өчүрүү |
| GET | `/boards/{id}/pins/` | Доскадагы пиндер |
| GET | `/boards/user/{user_id}/` | Колдонуучунун доскалары |

### Feed (`/feed`)
| Метод | Эндпоинт | Баяндама |
|---|---|---|
| GET | `/feed/` | Негизги лента |
| GET | `/feed/search/?q=...` | Издөө |
| GET | `/feed/tags/{tag_name}/` | Тег боюнча |
| GET | `/feed/trending/` | Trending |

### Комментарийлер (`/comments`)
| Метод | Эндпоинт | Баяндама |
|---|---|---|
| GET | `/comments/pin/{pin_id}/` | Пиндин комментарийлери |
| POST | `/comments/` | Комментарий жазуу |
| DELETE | `/comments/{id}/` | Комментарийди өчүрүү |

### Колдонуучулар (`/users`)
| Метод | Эндпоинт | Баяндама |
|---|---|---|
| GET | `/users/{id}/` | Колдонуучунун профили |
| GET | `/users/{id}/pins/` | Колдонуучунун пиндери |
| GET | `/users/{id}/saves/` | Сакталган пиндер |
| POST | `/users/{id}/follow/` | Подписка / алуу |
| GET | `/users/{id}/followers/` | Подписчики |
| GET | `/users/{id}/following/` | Подписки |

### Upload (`/upload`)
| Метод | Эндпоинт | Баяндама |
|---|---|---|
| POST | `/upload/` | Сүрөт жүктөө |

## Маалымат базасынын түзүлүшү

```
User (1) ──► Pin (N) ──► Comment (N)
  │             │
  │             ├──► Like (N)
  │             │
  │             └──► Save (N)
  │
  └──► Board (N) ──► Pin (N)

Pin ──► Tag (M:N)  (pin_tags таблицасы)

User (1) ──► Follow (N) ──► User (1)
(follower)              (followed)
```
